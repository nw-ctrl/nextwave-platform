# Supabase Project

Project name: `nextwave-core`

## Apply schema
Run SQL from `packages/database/supabase/schema.sql` in the Supabase SQL editor or via CLI migration.

Includes multi-repo registry tables:
- `github_connections`
- `github_repositories`

Includes multi-tenant billing tables:
- `billing_profiles`
- `subscription_entitlements`
